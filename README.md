# ✈️ British Airways Data Science Job Simulation

## 🚀 Overview
This project is part of the British Airways Data Science Job Simulation (Forage).  
It focuses on applying data science techniques to analyze customer data and generate insights that can support business decision-making in the airline industry.

---

## 🎯 Objective
To explore airline customer data, understand patterns in customer behavior, and apply data-driven approaches to improve business insights and decision-making.

---

## 🛠 Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Data Visualization (Matplotlib / Seaborn)
- Jupyter Notebook

---

## 📊 Tasks Performed

### 1. Data Understanding
- Loaded and explored structured datasets
- Identified missing values and data quality issues

### 2. Data Preprocessing
- Cleaned and prepared dataset for analysis
- Encoded categorical variables where necessary

### 3. Exploratory Data Analysis (EDA)
- Analyzed customer behavior patterns
- Visualized key trends using graphs and charts

### 4. Predictive Modeling (if applicable in your work)
- Built basic classification models to identify patterns in customer data
- Evaluated model performance using standard metrics

- Identified patterns in customer behavior
- Understood factors influencing customer decisions
- Demonstrated ability to apply data science workflow on real-world business scenario
